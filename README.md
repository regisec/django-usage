## Requirements
- Django: 1.8 or 1.9
- Python: 2.7, 3.4, 3.5

## Instalation
To install the **Django Usage** you can use `pip` as bellow

    pip install django-usage

Then add `'django_usage'` in your `INSTALLED_APPS` settings

    INSTALLED_APPS = (
        ...,
        'django_usage',
    )
